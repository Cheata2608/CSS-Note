The flex container becomes flexible by setting the display property to flex.

A Flexible Layout must have a parent element with the display property set to flex.

Direct child elements(s) of the flexible container automatically becomes flexible items.

1. The flex-direction Property

The flex-direction property defines in which direction the container wants to stack the flex items.

Specifies the direction of the flexible items inside a flex container.

A. The column value stacks the flex items vertically (from top to bottom).

flex-direction: column;

1
2
3

B. The column-reverse value stacks the flex items vertically (but from bottom to top).

flex-direction: column-reverse;

C. The row value stacks the flex items horizontally (from left to right).

flex-direction: row;

2. The flex-wrap Property

The flex-wrap property specifies whether the flex items should wrap or not.

Specifies whether the flex items should wrap or not, if there is not enough room for them on one flex line.

A. The wrap value specifies that the flex items will wrap if necessary.

flex-wrap: wrap;

3. The justify-content Property

The justify-content property is used to align the flex items.

Horizontally aligns the flex items when the items do not use all available space on the main-axis.

A. The center value aligns the flex items at the center of the container.

justify-content: center;

B. The flex-start value aligns the flex items at the beginning of the container (this is default).

4. The align-items Property

The align-items property is used to align the flex items.

Vertically aligns the flex items when the items do not use all available space on the cross-axis.

5. The align-content Property

The align-content property is used to align the flex lines.

Modifies the behavior of the flex-wrap property. It is similar to align-items, but instead of aligning flex items, it aligns flex lines.

In these examples we use a 200 pixels high container,
with the flex-wrap property set to wrap, to better demonstrate the align-content property.

A. The space-between value displays the flex lines with equal space between them.

align-content: space-between;
height: 100px;
flex-wrap: wrap;
overflow: scroll;
